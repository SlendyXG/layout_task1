# Новое русское вино

Сайт магазина авторского вина "Новое русское вино".

## Запуск

- Скачайте код
- Установите зависимости командой `pip install -r requirements.txt`
- Подготовьте Excel-файл с колонками: Название, Категория, Цена, Картинка (опционально: Сорт, Акция)
- Запустите сайт командой `python3 main.py`
- Перейдите на сайт по адресу [http://127.0.0.1:8080](http://127.0.0.1:8080).
- Для использования своего файла: `python3 main.py --data_file ваш_файл.xlsx`
- Для смены порта: `python3 main.py --port 8000`
